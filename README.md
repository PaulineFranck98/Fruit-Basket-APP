<h1 align="center">Fruit'n'Click</h1>

<br/>

<h3>📃 Description</h3>

<br/>

<strong>Fruit’n’Click</strong> est une application qui permet aux utilisateurs d'ajouter, consulter et gérer des fruits dans un panier.
